🚀 Pengenalan JavaScript

JavaScript adalah bahasa pemrograman yang digunakan untuk membuat halaman web menjadi interaktif dan dinamis.

Apa itu JavaScript?

JavaScript adalah bahasa pemrograman high-level, interpreted, dan dynamic yang memungkinkan Anda:

  • Membuat halaman web interaktif
  • Memanipulasi elemen HTML dan CSS
  • Menangani event user
  • Berkomunikasi dengan server

Cara Menulis JavaScript

// Console log - untuk menampilkan output console.log("Hello, World!"); // Alert - untuk menampilkan popup alert("Selamat datang di JavaScript!"); // Komentar satu baris menggunakan // /* Komentar multi baris */

🔢 Variabel & Tipe Data

Deklarasi Variabel

Ada 3 cara untuk mendeklarasikan variabel di JavaScript:

// var - cara lama (hindari penggunaan) var nama = "John"; // let - untuk variabel yang bisa berubah let umur = 25; // const - untuk konstanta (tidak bisa berubah) const PI = 3.14159;

Tipe Data Primitif

// String (teks) let nama = "Budi"; let pesan = 'Halo dunia!'; let template = `Nama saya ${nama}`; // Number (angka) let umur = 25; let tinggi = 175.5; // Boolean (true/false) let sudahMenikah = false; let aktif = true; // Undefined let belumDiisi; // Null let kosong = null;

Operator

// Operator Aritmatika let a = 10; let b = 3; console.log(a + b); // 13 (penjumlahan) console.log(a - b); // 7 (pengurangan) console.log(a * b); // 30 (perkalian) console.log(a / b); // 3.33 (pembagian) console.log(a % b); // 1 (modulus/sisa bagi) // Operator Perbandingan console.log(a > b); // true console.log(a == b); // false console.log(a === b); // false (strict equality) // Operator Logika console.log(true && false); // false (AND) console.log(true || false); // true (OR) console.log(!true); // false (NOT)

⚙️ Fungsi (Functions)

Deklarasi Fungsi

// Function Declaration function sapa(nama) { return "Halo, " + nama + "!"; } // Function Expression const tambah = function(a, b) { return a + b; }; // Arrow Function (ES6) const kali = (a, b) => a * b; // Arrow Function dengan satu parameter const kuadrat = x => x * x;

Parameter & Return

// Fungsi dengan parameter default function perkenalan(nama = "Anonymous", umur = 0) { return `Nama: ${nama}, Umur: ${umur} tahun`; } // Fungsi dengan multiple return function cekUmur(umur) { if (umur >= 18) { return "Dewasa"; } else if (umur >= 13) { return "Remaja"; } else { return "Anak-anak"; } } // Fungsi rekursif function faktorial(n) { if (n <= 1) return 1; return n * faktorial(n - 1); }

📦 Objek & Array

Objek (Objects)

// Membuat objek const mahasiswa = { nama: "Alice", umur: 20, jurusan: "Informatika", ipk: 3.75, // Method dalam objek perkenalan: function() { return `Nama saya ${this.nama}, umur ${this.umur}`; } }; // Mengakses properti console.log(mahasiswa.nama); // "Alice" console.log(mahasiswa["umur"]); // 20 // Menambah properti baru mahasiswa.semester = 5; // Object destructuring const {nama, umur} = mahasiswa;

Array

// Membuat array const fruits = ["apel", "pisang", "jeruk"]; const numbers = [1, 2, 3, 4, 5]; // Method array yang berguna console.log(fruits.length); // 3 fruits.push("mangga"); // menambah di akhir fruits.pop(); // menghapus yang terakhir fruits.unshift("anggur"); // menambah di awal fruits.shift(); // menghapus yang pertama // Array methods advanced const doubled = numbers.map(num => num * 2); // [2,4,6,8,10] const even = numbers.filter(num => num % 2 === 0); // [2,4] const sum = numbers.reduce((acc, num) => acc + num, 0); // 15 // Array destructuring const [first, second, ...rest] = fruits;

🎨 DOM Manipulation

Selecting Elements

// Memilih elemen berdasarkan ID const element = document.getElementById("myId"); // Memilih elemen berdasarkan class const elements = document.getElementsByClassName("myClass"); // Memilih elemen berdasarkan tag const paragraphs = document.getElementsByTagName("p"); // Query selector (modern way) const el = document.querySelector(".myClass"); // pertama yang ditemukan const els = document.querySelectorAll(".myClass"); // semua yang ditemukan

Mengubah Content & Style

// Mengubah text element.textContent = "Text baru"; element.innerHTML = "<b>Text dengan HTML</b>"; // Mengubah style element.style.color = "red"; element.style.backgroundColor = "yellow"; element.style.fontSize = "20px"; // Mengubah class element.classList.add("newClass"); element.classList.remove("oldClass"); element.classList.toggle("active"); // Mengubah atribut element.setAttribute("data-id", "123"); element.getAttribute("data-id");

👆 Event Handling

Jenis-jenis Event

// Click event button.addEventListener('click', function() { console.log('Button diklik!'); }); // Mouse events element.addEventListener('mouseenter', () => { console.log('Mouse masuk'); }); element.addEventListener('mouseleave', () => { console.log('Mouse keluar'); }); // Keyboard events document.addEventListener('keydown', (event) => { console.log('Key pressed:', event.key); }); // Form events form.addEventListener('submit', (e) => { e.preventDefault(); // mencegah form submit console.log('Form disubmit'); });

Event Object

button.addEventListener('click', function(event) { console.log('Event type:', event.type); console.log('Target element:', event.target); console.log('Mouse position:', event.clientX, event.clientY); // Mencegah behavior default event.preventDefault(); // Menghentikan propagasi event event.stopPropagation(); });
